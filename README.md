# Software testing project

## Description
In this project an e-commerce utility library is being tested using unit tests and GitHub Actions workflow. 
This project runs automatic tests on each code push to the repository. Unit tests were performed for 10 prioritized functions.

## Testing the project
Run project tests and coverage report with this command:
`npm test`
